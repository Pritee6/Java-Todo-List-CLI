# Java CLI To-Do List Application

A simple, interactive command-line interface (CLI) application built in Java to manage daily tasks. This project demonstrates core Java concepts like the Collections Framework and input validation.

## Features
- **Add Tasks:** Dynamically insert new tasks into an ArrayList.
- **View Tasks:** Check all current tasks in your ArrayList.
- **Remove Tasks:** Delete tasks from the ArrayList using index positions.
- **Input Validation:** Gracefully handles invalid inputs (strings/characters) when numerical selections are expected without crashing the application.

## Technologies Used
- Java SE (Standard Edition)
- `java.util.ArrayList`
- `java.util.Scanner`
